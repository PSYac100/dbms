안녕하세요 국토정보교육원에 박소영입니다.

저는 공간정보 분야, 특히 WebGIS 쪽을 강의하고 있습니다. 올해부터는 교육생분들이 더 효과적으로 학습하실 수 있도록 웹 사이트를 하나 만들었습니다. 저는 실습교육 중 강의 내용 학습 및 복습을 위해 연습 문제 또는 퀴즈를 제시합니다. 교육생 여러분들이 자가 학습을 통해 습득하시되 본 사이트에서 해당 문제에 대한 정답을 확인하시고, 향후 학습 시 참고용으로 활용하시면 될 것 같습니다.

계속해서 기술이 발전하고 여러 학문이 융합하는 시대에 새로운 것을 배우는 것은 분명 어렵고 힘든 일입니다. 그러나 새로운 것을 하나씩 알아가는 재미도 있지 않을까요? 저도 가능했기에 여러분도 충분히 하실 수 있으실 겁니다. 화이팅!!

02. 도로 중 6차선 이상인 도로를 찾아 길이를 구하시오.


SELECT lanes, ST_LENGTH(geom/1000) AS length
FROM road
WHERE lanes >= 6

03. 서울특별시 읍면동에서 area라는 칼럼을 삽입하여 모든 행정구역의 면적을 구하고, 면적이 5km2 이상인 행정구역을 찾으시오.


ALTER TABLE dong ADD COLUMN area real;
UPDATE dong SET area = ST_AREA(geom);
SELECT * FROM dong
WHERE area > 5.0;

04. 서울특별시의 시군구별 면적을 계산하고, 면적이 가장 큰 상위 5개를 선택하여 나타내 봅시다.


SELECT sig_kor_nm AS name, ST_AREA(geom)/1000000 AS area
FROM sigungu 
ORDER BY area DESC

05. 도로 중 강남구 필지를 가로질러서 지나가는 2차선 이상인 도로를 찾아 봅시다.


SELECT j.pnu, r.lanes, r.road_name
FROM jijuk AS j, road AS r
WHERE ST_CROSSES(j.geom, r.geom)

06. 37.9428, 126.97653 지점을 기준으로 반경 1km 이내에 있는 마트를 찾아 봅시다.


SELECT * 
FROM mart 
WHERE ST_DISTANCE(ST_GeomFromText('Point(126.97653 37.59428)', 4326), 
mart.geom) < 1000

WHERE ST_DWITHIN(ST_GeomFromText('Point(126.97653 37.59428)', 4326), 
mart.geom, 500)

WHERE ST_INTERSECTS(mart.geom, ST_BUFFER(ST_GeomFromText('Point(126.97653 37.59428)', 4326), 500))

07. 병원 중 강남구에 위치한 병원을 찾아 봅시다.


SELECT * 
FROM hospital

09. 6차선 이상의 도로에 인접한 필지에 건설된 건물의 기하 데이터를 구해 봅시다.


SELECT j.pnu, j.jimok, b.geom
FROM road AS r, jijuk AS j JOIN building AS b ON j.pnu = b.pnu
WHERE ST_INTERSECTS(r.geom, j.geom) AND r.lanes >= 6

10. 토지 소유자가 개인이면서 공유인수가 10명 이상인 필지를 찾아 면적을 구해 봅시다.


SELECT j.pnu, o.소유구분, o.공유인수, ST_AREA(j.geom) AS area
FROM jijuk AS j JOIN owner AS o ON j.pnu = o.pnu
WHERE o.소유구분 = '개인' AND 공유인수 >= 10

11. 토지 소유자가 개인이면서 공유인수가 10명 이상인 필지를 찾아 면적을 구해 봅시다.
