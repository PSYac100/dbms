2. 서울특별시 동대문구의 행정 경계의 면적은 얼마인가?(단, 단위는 km2으로 표현)

SELECT sig_kor_nm AS name, ST_AREA(geom)/1000000 AS area
FROM sigungu
WHERE sig_kor_nm='동대문구';

3. 한강과 인접한 서울특별시의 구는?

SELECT sig_kor_nm
FROM sigungu, hanriver 
WHERE cd=1 and ST_INTERSECTS(sigungu.geom, hanriver.geom)

4. PNU, 주소, 지목, 공시지가(2024년 기준) 정보를 가지는 테이블을 생성해 봅시다.

CREATE TABLE price1 AS
SELECT j.pnu AS pnu, j.address AS 주소, j.jimok AS 지목, p.price24 AS 공시지가
FROM jijuk AS j JOIN price AS p ON j.pnu = p.pnu

5. 지가의 상승률이 높은 상위 10개의 필지와 각 필지의 면적은?

SELECT j.pnu AS pnu, j.jimok AS 지목, ST_AREA(j.geom) AS 면적, (p.price24 – p.price23)/p.price23*100 AS 지가상승률
FROM jijuk AS j JOIN price AS p ON j.pnu = p.pnu
WHERE p.price23 != 0 AND p.price24 != 0
ORDER BY 지가상승률 DESC LIMIT 10

6. 서울특별시 서초구의 건물이 건설된 필지의 지목별 필지의 수는?

SELECT j.jimok, COUNT(j.jimok) AS count
FROM jijuk AS j JOIN building AS b ON j.pnu = b.pnu
GROUP BY j.jimok
ORDER BY count ASC
