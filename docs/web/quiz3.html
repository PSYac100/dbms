01. 박국토의 하루 지출 내역이 다음과 같다고 할 때, 하루 지출 비용의 합계를 구한 후 적정 지출 비용의 초과 여부를 출력해 봅시다.

let traffic = 3000;
let food = 8000;
let drink = 6000;
let result = traffic + food + drink;
result = result > 10000? result-10000 + "원 초과" : "돈 관리 참 잘했어요!";
document.write(result);
